# Customer Segmentation Dashboard

## 📊 Project Overview
This project analyzes customer data to identify spending patterns using Excel, SQL, and Power BI.

## 🛠 Tools Used
- Excel
- SQL (SQLite)
- Power BI

## 🔍 Key Insights
- Female customers show slightly higher spending
- Certain age groups have higher spending scores
- Income vs Spending reveals diverse customer behavior
